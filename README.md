### Installing

A step by step series of examples that tell you how to get a development env running

Say what the step will be
With npm 
```
npm install
```

With yarn 
```
yarn install
```

To run on local with npm
```
npm run dev 
```

To run on local with yarn
```
yarn dev 
```

App will run on

```
Local: http://localhost:8080/
```
